# hint 配置说明

## HTMLHint

### 安装

```
$ npm install htmlhint -g
```

## csscomb

```
$ npm install csscomb -g
```

## eslint

### 安装

`_eslintrc_es7`配置文件为了支持最新的 ES7 语法和 react开发，使用了 babel-eslin parser 和 eslint-plugin-react 插件。

```
$ npm install babel-eslint -g
$ npm install eslint eslint-plugin-react -g
```
