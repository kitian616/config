# zsh配置说明

## install zsh

### macOS

```
brew install zsh zsh-completions
```

### Ubuntu, Debian

```
apt install zsh
```

### Arch Linux

```
pacman -S zsh
```

## install [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh)

```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
```
